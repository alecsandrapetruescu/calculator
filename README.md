## Simple Calculator
Command line application with Spring Boot

### How to use
> mvn clean compile package

> java -jar calculator-0.0.1-SNAPSHOT.jar 1 + 1

> java -jar calculator-0.0.1-SNAPSHOT.jar 1 - 1

> java -jar calculator-0.0.1-SNAPSHOT.jar 1 * 2

> java -jar calculator-0.0.1-SNAPSHOT.jar 1 / 2
